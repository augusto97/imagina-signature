Preguntas frecuentes

¿Las firmas funcionan en Outlook?

Sí. La compilación produce HTML basado en tablas con CSS inline y comentarios condicionales para Outlook (vía MJML).

¿Las imágenes se incrustan en cada email?

No. Se enlazan al storage configurado (Media Library o S3). Los clientes de correo descargan la imagen del servidor.

¿Necesito un servicio externo?

No para Media Library. Para S3 sí, pero usas el proveedor que tú elijas; el plugin no llama a servicios SaaS del autor.

¿Funciona sin Composer en producción?

Sí. El autoloader es propio. Composer sólo se usa en desarrollo (linters y tests).
