Imagina Signatures

Storage

Imagina Signatures soporta dos modos de almacenamiento de assets (imágenes que aparecen en las firmas):

  • Media Library: las imágenes se guardan en wp-content/uploads/imagina-signatures/. Sin configuración, ideal para empezar.
  • S3-compatible: presigned uploads directamente al bucket que elijas. Soportamos Cloudflare R2, Bunny, Amazon S3, Backblaze B2, DigitalOcean Spaces, Wasabi, MinIO y endpoints custom.

Las credenciales se cifran con AES-256-CBC usando una clave derivada de AUTH_KEY antes de guardarse en la base de datos.
