Installing your signature: gmail

This page is a stub. Step-by-step screenshots for each major client (with the exact menu paths to paste an HTML signature) ship with the public 1.0 release.

The general flow: in the editor click Export HTML → Copy HTML. Then paste into your client's signature editor in HTML / source mode, not the WYSIWYG view (most clients lose inline styles when pasted into the rich-text mode).
