Editor guide

Adding blocks

Click any block in the left sidebar to append it at the end of your signature, or drag it onto the canvas to drop it at a specific position.

Editing

Click a block on the canvas to select it. The right sidebar swaps to show that block's properties (text, colour, padding, etc.). Editing happens in the sidebar — never directly on the canvas — so cursor and selection behave predictably.

Reordering and deleting

The floating toolbar above each selected block has duplicate, drag, and delete buttons. You can also drag any block to a new position.

Keyboard shortcuts

  • ⌘/Ctrl + Z — Undo
  • ⌘/Ctrl + Shift + Z — Redo
  • ⌘/Ctrl + D — Duplicate selected block
  • Backspace / Delete — Delete selected block
  • Esc — Clear selection

Saving

Changes auto-save 1.5 seconds after you stop editing. The topbar shows the current status: Saved, Saving…, or Unsaved changes.

Exporting

Click Export HTML in the topbar. The compiler runs and shows the email-safe HTML alongside any warnings (Gmail clipping, missing alt attributes, etc.). Use Copy HTML to put it on the clipboard, or Download .html to save it as a file.

Preview

Click Preview to see the signature rendered the way Gmail / Outlook / Apple Mail approximately render it. Real client testing still belongs in the actual mail client.
