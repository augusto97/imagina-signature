Imagina Signatures

Instalación

Requisitos

  • WordPress 6.0 o superior
  • PHP 7.4+ (testeado en 7.4, 8.0, 8.1, 8.2, 8.3)
  • MySQL 5.7+ / MariaDB 10.3+

Pasos

  1. Descarga el ZIP imagina-signatures-x.y.z.zip desde tu cuenta o el repositorio.
  2. En tu sitio WordPress entra a Plugins → Añadir nuevo → Subir plugin.
  3. Selecciona el ZIP y pulsa Instalar ahora.
  4. Activa el plugin.
  5. Sigue al asistente de configuración (Setup wizard).
Tras la activación se crean las tablas imgsig_*, los planes por defecto y las plantillas del sistema.
