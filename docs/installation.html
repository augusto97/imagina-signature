Installation

  1. Download the latest imagina-signatures-x.y.z.zip release.
  2. In wp-admin, go to Plugins → Add New → Upload Plugin.
  3. Pick the ZIP and click Install Now, then Activate.
  4. You'll find Imagina Signatures in the left sidebar.

Requirements

  • WordPress 6.0+
  • PHP 7.4+ with openssl, mbstring, json
  • MySQL 5.7+ / MariaDB 10.3+

Storage

Out of the box the plugin uses the WordPress Media Library — no configuration needed. To switch to an S3-compatible backend, see the storage guides.
