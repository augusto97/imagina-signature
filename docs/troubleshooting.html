Solución de problemas

Las imágenes no se ven en Gmail/Outlook

Verifica que la URL pública del asset es accesible sin autenticación. En modo S3, comprueba el ACL del bucket o el CDN configurado.

Aparece un aviso de "HTML excede 102 KB"

Gmail clipea mensajes mayores a 102 KB. Reduce el número de bloques o el tamaño de las imágenes embebidas.

El editor no carga

Vacía el cache del navegador y revisa la consola — el bundle reporta errores con un Error Boundary global.

Sólo veo "Sign in" al activar el plugin

El plugin requiere PHP 7.4 o superior. Si tu hosting está por debajo, actualiza la versión de PHP.
